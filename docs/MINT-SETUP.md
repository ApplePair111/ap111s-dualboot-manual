# Setup Linux!

## Step 1. Setup

OK. So if you're in a desktop environment, good. So, click Install Linux Mint.
Now, just follow the instructions.

IMPORTANT: At the end, ALWAYS choose Install with Windows Boot Manager. Otherwise, Windows can get wiped.

## Step 2. Boot

Now, restart AGAIN. When you arrive at windows, just click Restart with shift AGAIN.

When you enter the blue screen, choose Troubleshoot, then Advanced Options. 

Then, click UEFI Firmware Settings.
