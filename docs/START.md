# Welcome to this chapter!

This chapter will teach you to boot the OS.

## Step 1. Get the ISO

So, you need to download the ISO. Find it [here](https://linuxmint.com/download.php)

Did you download it? Well if you did, let's move forward.

You need a USB. Make sure no important data cuz it WILL BE WIPED.

## Step 2. Flash and boot the USB

If you don't already have it, get [Rufus](https://rufus.ie).

Once you have it open, click Device and put your USB in the dropdown.

Then, choose the Linux Mint ISO you got as the SELECT button in the Boot Selection. 

KEEP EVERYTHING ELSE AS-IS

Now, what you want to do is to go to the start menu and HOLD SHIFT while clicking the Restart button. When the blue screen appears, click "Use a Device".

Select your USB. If done correctly, you'll enter a screen with a big LM on the screen. If so, good, keep going.
